# lolly-claude-plugin

Claude cowork plugin that packages the Lolly ERP agent as a small, semantic MCP tool surface (`lolly_query`, `lolly_report`, `lolly_task_status`) with Zod-validated response envelopes and skills teaching cowork how to judge intent-vs-output.

See `docs/` for architecture, envelope, tool, and deploy references.
