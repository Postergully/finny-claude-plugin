# finny-claude-plugin

Claude cowork plugin that packages the Finny ERP agent as a small, semantic MCP tool surface (`finny_query`, `finny_report`, `finny_task_status`) with Zod-validated response envelopes and skills teaching cowork how to judge intent-vs-output.

See `docs/` for architecture, envelope, tool, and deploy references.
